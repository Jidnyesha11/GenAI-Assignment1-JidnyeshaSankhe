## Description
This assignment demonstrates the use of Python data structures including lists, tuples, sets, and dictionaries. It also combines these concepts to perform operations on product data using a Jupyter Notebook (`.ipynb`).

---

## Tasks Completed

### Task 1 - Lists and Tuples
- Created and modified a list of products.
- Converted a list into a tuple.
- Performed basic list operations such as adding and removing elements.

### Task 2 - Sets
- Created a set of product categories.
- Added new categories.
- Checked membership.
- Counted the total number of unique categories.

### Task 3 - Dictionaries
- Created a dictionary of product prices.
- Added, updated, and removed dictionary entries.
- Calculated the average, maximum, and minimum product prices.

### Task 4 - Combined Operations
- Created a catalog as a list of tuples containing product names and prices.
- Created a dictionary mapping categories to product names.
- Identified the category with the maximum number of products.
